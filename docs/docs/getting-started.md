## What is this?

High-level overview of the app and these docs.
Use this page as a Readme.md

## Requirements
- Node 20+
- npm
- .env file (ask an admin)

## Install & run (app project)
```bash
npm ci
npm start
```

## Run tests
```bash
npm test
```

## Run Storybook (components)
```bash
npm run storybook
```

## Docs site (Docusaurus)
```bash
cd docs
npm ci
npm start
```
